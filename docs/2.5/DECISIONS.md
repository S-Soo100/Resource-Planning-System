## 확정 결정 사항

### 1. 재구매 예정 고객 표시 위치
- **결정**: 별도 목록 페이지 + 메인 대시보드 위젯
- **구현**: `/repurchase` 페이지 + MainMenu 상단 알림 배너 (N명 표시, 클릭 시 이동)

### 2. 고객 서류 접근 위치
- **결정**: 발주 상세 + 고객 목록 양쪽에서 접근 가능
- **구현**: 발주 상세 페이지 + UserEditModal에 CustomerDocumentSection 추가

### 3. 출고완료 건 수정 UI
- **결정**: 기존 DetailsEditModal 확장
- **구현**: 환급/입금/세금계산서 필드 추가, 환급 해당없음 시 다른 필드 비활성화

### 4. 주민등록번호 마스킹
- **결정**: readOnly 모드에서 `900101-*******` 형태로 마스킹
- **구현**: UserEditModal에서 isReadOnly 시 정규식 마스킹 적용

### 5. 세금계산서 표시 위치
- **결정**: 발주 상세 페이지에 세금계산서 섹션 추가
- **구현**: 파일 업로드 + 목록 + 삭제 기능

### 6. 입금 상태 UI
- **결정**: Select 드롭다운 (5개 옵션)
- **구현**: 자부담금/전액/선금/중도금/잔금

### 7. B2C/B2B 고객 분류 관리
- **결정**: 재구매 예정 고객 페이지에 분류별 필터 탭 + 엑셀 내보내기
- **구현**: 전체/B2C/B2B/미분류 필터, 필터 적용 상태로 엑셀 다운로드

### 8. 수급자 여부 추가 로직
- **결정**: 불필요 — 체크박스 표시만으로 충분
- **구현**: 현행 유지
