## 데이터 매핑

### 필요한 데이터 필드
- **발주 정보**:
  - `id`: 발주 ID
  - `title`: 발주 제목
  - `purchaseDate`: 발주일자
  - `deliveryRequestDate`: 납품요청일
  - `status`: 발주 상태

- **공급받는자 정보**:
  - `supplierName`: 판매처명
  - `supplierContact`: 담당자 정보
  - `deliveryAddress`: 배송지 주소
  - `recipientName`: 수령인

- **품목 정보**:
  - `orderItems`: 품목 배열
    - `itemName`: 품목명
    - `quantity`: 수량
    - `sellingPrice`: 단가 (판매 단가)
    - `vat`: 부가세 (단가 × 수량 × 10%)
    - `totalPrice`: 금액 (단가 × 수량 + 부가세)

- **금액 정보**:
  - `totalPrice`: 총 거래금액
  - `itemCount`: 품목 종류 수
  - `totalQuantity`: 총 수량

## 스타일링 가이드

### 색상
- **헤더**: 파란색 배경 (#3B82F6) + 흰색 텍스트
- **테이블 헤더**: 회색 배경 (#F3F4F6)
- **테이블 테두리**: 회색 (#E5E7EB)
- **강조 텍스트**: 파란색 (#2563EB)

### 폰트
- **제목**: 24px, Bold
- **섹션 헤더**: 16px, Semi-bold
- **본문**: 12px, Regular
- **테이블**: 11px, Regular

### 여백
- **페이지 여백**: 상하좌우 20mm
- **섹션 간격**: 15px
- **줄 간격**: 1.5

---

## 출력 형식
- **페이지 크기**: A4 (210mm × 297mm)
- **방향**: 세로
- **파일명 형식**: `거래명세서_[판매처명]_[발주번호]_[발행일자].pdf`
  - 예: `거래명세서_서울병원_ORD-12345_20260209.pdf`

## 추가 기능 제안
1. **QR 코드**: 발주 상세 페이지로 이동할 수 있는 QR 코드 추가
2. **전자서명**: 담당자 전자서명 기능
3. **워터마크**: "원본" 또는 "사본" 워터마크 표시
4. **다국어 지원**: 영문 거래명세서 옵션
5. **이메일 발송**: PDF 생성 후 자동 이메일 발송 기능
